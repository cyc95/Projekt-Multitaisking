Implementierung der Funktion int _read( int fd, char * buf, int cnt ).

Diese Datei implementiert die Funktion _read( ) (und weitere, benötigte Funktionalitäten), sodass die Funktionen scanf( ), getchar( ) und gets( ) aus stdio.h. von der serielle Schnittstelle lesen.
