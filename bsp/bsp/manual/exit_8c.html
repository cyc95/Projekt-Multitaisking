Implementierung der Funktion void _exit( int err ).

Die Funktion _exit( ) wird sowohl in newlib als auch in optlib aus exit( ) herraus aufgerufen. Mit der Implementierung von _exit( ) kann exit( ) (stdlib.h) verwendet werden.
