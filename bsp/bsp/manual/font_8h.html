Installation einer weiteren Schriftart
Um eine weitere Schriftart zu installieren müssen Sie in dieser Datei (font.h) einen weiteren define a la
#define fontUSE_FONT_<here-goes-your-font-name>
vorsehen und die übrigen auskommentieren. Anschließend müssen Sie die Konstante fontWIDTH auf die Breite ihrer Schriftart ändern und in font.c die Daten der Schriftart in font[][] hinterlegen. Beachten Sie hierbei, dass eine Schriftart, die höher als 8 Pixel ist ohne weitere Änderungen zu Problemen führt. Beachten Sie bei der Datenhinterlegung in font[][] außerdem die Pixeladressierung des LCD. Eine Möglichkeit diese nachzuvollziehen ist der Aufruf von lcdPrintTestPattern( ).

Definiert in Datei font.h.

Makro-Dokumentation

#define fontUSE_FONT_8X8

Mit diesem/n define können Sie steuern welche Schriftart verwendet werden soll.

Zu beachten
Stellen Sie stets sicher, dass immer genau einer der der Defines nicht auskommentiert ist.

Variablen-Dokumentation

const unsigned char font[256][ 8 ]

Das ist das Array, dass die Schriftart hält. Bspw. liegen die für den Buchstaben 'H' relevanten Daten an der Stelle &font[ 'H' ][ 0 ]. Die Daten sind spaltenweise (vertikal) abgelegt. Das LSB liegt in der obersten Zeile eines Buchstaben/ eines Zeichens.
