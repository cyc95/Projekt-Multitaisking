Definiert in Zeile 206 der Datei led.h.

#define ledInitialize ( )

Dieses Makro initialisiert die den LEDs 4 bis 15 zugeordneten Register.

Es werden die entsprechenden Bits der Port-Output-Data-Register (PODR) der Ports D und E mit 1en belegt, sodass die zwölf LEDs mit dem Zustand 'aus' initialisiert werden. Anschließend werden die entsprechenden Bits der Port-Direction-Register (PDR) der Ports D und E mit 1en belegt, sodass die LED-Pins als Ausgänge fungieren und nun mit den Makros ledToggle(), ledTurnOn() und ledTurnOff() genutzt werden können.

#define ledToggle (   x)
Wert:
{ \
\
/*
* den wert des bits im PODR invertieren
* (siehe LED4, LED5, ..., LED15)
*/ \
(x) ^= 1; \
\
}

Dieses Makro invertiert den Zustand der LED x.

Parameter
xLED deren Zustand invertiert werden soll. Nutzen Sie zwingend LED4, LED5, ..., LED15.
Zu beachten
Nutzen Sie zwingend LED4, LED5, ..., LED15 als Argument!
Vorbedingung
Rufen Sie zu Beginn ihres Programms einmal ledInitialize() auf. Erst dann funktioniert dieses Makro wie erwartet.
Beispiel
#include "led.h"
void main(void) {
ledInitialize();
ledToggle(LED4);
}
